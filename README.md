# Project Nhóm 3 - CS 403 E

## Technology

### Backend

- ✅ NodeJS (Express)
- ✅ Database with MySQL

### Frontend

- ✅ ReactJS
- ✅ UI Library with shacdn-ui
- ✅ Deploy on Vercel
- ✅ TailwindCSS

## Start project

```
git clone https://github.com/vophuocthanh/app-cs403.git
cd /backend && setup .env && npm install && npm start
cd /frontend && pnpm install && pnpm run start
```
